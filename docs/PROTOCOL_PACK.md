# FIFA volunteer protocol pack (all stadiums)

Generated by `src/lib/protocol-pack.ts` and seeded for **every** venue.

**Count:** 27 topics × 6 stadiums = **162** protocol records (landmarks/gates differ per stadium).

Used by:

- Volunteer chat **Category A** (FAQ answers)
- Volunteer chat **Category D** (emergency SOP deploy)
- Fan Voice Assist (spoken facility answers)

---

---

## Category B (not in this pack)

City tourism / outside-the-stadium questions (e.g. “best restaurants downtown”) are **out of scope** → fixed Fan Guide rejection. In-stadium food and facilities stay **Category A**.

---

## After pull / protocol code change

Reload the seed so new pack content is in the database:

```bash
# local
curl -X POST http://localhost:3000/api/admin/reset

# production (Render URL)
curl -X POST https://YOUR-SERVICE.onrender.com/api/admin/reset
```

Or locally: delete `data/db.json` and restart `npm run dev` (file backend only).
